## Beacon HQ Visual Style Reference

**Brand Colors:**
- Primary Background: #0a0e27 (Dark Navy)
- Primary Accent: #00d4ff (Bright Cyan)
- Secondary: #1a0d2e (Deep Purple)
- Highlight: #00fff5 (Teal)
- Text: #ffffff (White), #e0e0e0 (Light Gray)

**Typography:**
- Headers: Modern sans-serif (Inter, Roboto, SF Pro)
- Bold weights for impact
- Clean, technical appearance

**Visual Elements:**
- Geometric shapes (shields, hexagons, grids)
- Data flow visualizations
- Terminal/console aesthetics
- Configuration file representations
- Checkmarks and validation symbols

**Mood Keywords:**
- Protective
- Professional
- Technical
- Trustworthy
- Developer-focused
- Security-oriented

---

## Usage Notes

1. **Generate in Order:** Title Card → Logo Mark → Horizontal → App Icon
2. **Maintain Consistency:** Use same colors and style across all assets
3. **Test at Sizes:** Ensure icons work at small sizes (32px, 64px)
4. **Export Formats:** PNG with transparency for logos, JPEG for title cards
